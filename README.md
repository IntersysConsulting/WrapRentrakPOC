# WrapRentrakPOC
POC for different View grid libraries

## Requirements
- Windows
- MySQL
- VS 2015

## Setup Instructions
- Restore `rentrak` DB from zip file in DataDump dir. Make sure stored procs are also restored.
- Update DB connection string in Wed.config with proper creds.
- Run project in VS.
